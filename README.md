## Installation

Clone the repository:

```sh
git clone <repository_url>
cd PROJECT_NAME
```

Install dependencies using pnpm:

```sh
pnpm i
```

Start the development server:

```sh
pnpm run dev
```

Access the application at http://localhost:3000.

Run tests using Jest:

```sh
pnpm run jest
```

Run tests using Cypress:

```sh
pnpm run cypress
```
